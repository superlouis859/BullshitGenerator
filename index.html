帮助翻译为其他的语言版本

关于Vistalizator

22 April 2012

你曾在国外买过Windows的笔记本电脑并且希望运行你的母语版本吗?

试试 Vistalizator - 这个工具让你轻松更换Windows系统的显示语言，即使是Windows初级版，家庭普通版，家庭高级版和专业版（微软官方仅提供旗舰版和企业版更换语言的功能）.

Vistalizator 突破了非旗舰版Windows系统中“多语言用户界面MUI”只有一种语言的限制.
请看细节 在Windows中使用多语言.

功能特性

  • 适用于32位和64位Windows Vista/Windows 7
  • 5分钟之内更换Windows显示语言，妥妥的！(快速安装模式)
  • 使用微软官方语言包Windows语言包
  • 你可以安装全部36个 多语言用户界面MUI 语言并且在他们之间轻松切换
  • 你可以在特定的MUI之上再安装55个Windows Vista 语言界面包LIP 语言
  • 你可以在特定的MUI之上再安装59种Windows 7 语言界面包LIP 语言
  • 兼容Vista Service Pack 1/2 和 Windows 7 Service Pack 1
  • 符合用户账户控制 (UAC)安全性 - 提示需要管理员权限
  • Windows 语音识别 功能在其所有支持的语言上工作正常:
    中文，英文，法文，德语，日语和西班牙语
  • 可以用 内置Internal 或者 快速Express模式安装语言, 请看 细节

Windows的多语言系统

可用如下方法获得不同的/多种显示语言的用户界面:

  • 你很幸运如果你是Windows旗舰版/企业版的用户
  • 升级你的版本到旗舰版 - 购买 Windows Anytime Upgrade
  • 用其他语言的Windows安装DVD完全重新安装一个Windows
    (你可以用 Windows Easy Transfer 来转移你的数据和配置)
  • 制作你自己的多语言Windows DVD 并且完全重新安装一个Windows
  • 在Windows Vista 或 Windows 7中运行 Vistalizator 安装一种语言只需几分钟！
